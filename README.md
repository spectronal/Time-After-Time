# Time After Time

> *Cada segundo é seu. E cada segundo se vai.*

## Sobre

Time After Time é um jogo incremental simples, mas com uma proposta diferente: ele quer te fazer pensar no que você está "clicando" por aí enquanto o tempo passa. Cada click representa um momento que não volta. Você pode investir esses momentos em upgrades — mas no fim, o tempo é o mesmo que escorre.

No começo é só um botão. Depois você percebe que o tempo está se movendo sozinho. E aí você quer mais. E mais. Mas a sensação nunca muda: *está sempre passando*.

## Como jogar

1. Clique em **"One more"** para ganhar "segundos"
2. No painel da direita, troque seus segundos por upgrades:
   - **Momentum** — o tempo passa sozinho, sem você clicar
   - **Déjà vu** — cada click vale o dobro
   - **Sprint** — pressiona o acelerador, +5 por click
   - **Fleeting** — +5 segundos automáticos por tick
3. Use as **tabs** para filtrar entre upgrades automáticos e por click
4. Clique em **Save** para salvar seu progresso (ou espere o auto-save a cada 30s)

## Como rodar

Basta abrir o arquivo `index.html` no navegador. Não precisa de servidor — é tudo HTML, CSS e JavaScript puro.

## Tecnologias

- **HTML5** — estrutura
- **CSS3** — visual (sem frameworks)
- **JavaScript (vanilla)** — toda a lógica do jogo, sem bibliotecas externas
- **localStorage** — salvamento local no navegador
